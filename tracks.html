Tracks
Broadcasting

Course Overview

All students take a Radio and TV broadcasting course at CHS. CHS has a newly renovated tv and radio studio with state of the art technology. Students are able to explore this media through morning news and creating their own shows. Click to learn more.


Courses Offered

  • - Introduction to TV/Radio
  • - Live Studio Production-Radio/TV
  • - Advanced Studio Production-TV
  • - Advanced Studio Production-Radio
Visual Arts

Course Overview

CHS requires two introductory design courses before branching into specified interests. This includes traditional art, digital art, animation, and photography. Students become proficient in using many Adobe Suite programs. Click to learn more.


Courses Offered

  • - Communications Technology
  • - Introduction to Design
  • - Animation
  • - Graphic Arts
  • - Illustration & Design
  • - Photography
  • - Advertising Design
  • - AP Studio Art

Digital Video

Course Overview

In this track, students learn the filming process from creating following the steps to come up with an idea, shoot the project, and edit it together. Advanced courses further film production skills and promote creativity in the classroom setting. Click to learn more.


Courses Offered

  • - Digital Video
  • - Digital Video II
  • - Modern History Through Film
  • - Film Studies
  • - Advanced Digital Video

Journalism

Course Overview

Students learn and analyze current events and use their own writing skills to develop different stories and angles. CHS requires an introductory class within the journalism field. Students write and analyze media and pursue this track by taking Advanced Mass Media and/or Public Relations. Click to learn more.


Courses Offered

  • - Mass Media
  • - Public Relations
  • - Visual Mass Media
  • - Advanced Mass Media

Programming

Course Overview

Students are able to learn multiple coding languages to create personalized projects. All students take an introductory course to learn the basics of Python. Students can further their coding knowledge by learning Javascript and HTML/CSS in the advanced courses. Click to learn more.


Courses Offered

  • - Introduction to Programming
  • - Java Programming
  • - Advanced Java
  • - Web Design & Social Media Marketing
  • - Advanced Web Design
