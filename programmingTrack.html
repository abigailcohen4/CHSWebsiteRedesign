Introduction to Programming

Every CHS student in their sophomore year takes a mandatory introduction course to computer science where students learn the fundamentals of Python. The classroom has a full set of Macs for students to use. The class meets everyday for one marking period. Students learn the language through powerpoints and mini projects. The course focuses on learning variables, functions, and their behaviors together. Students reflect their understanding from the marking period by coding Go Fish, Hangman, and an interactive game of their choosing. All computer science courses are taught by Ms. Laura Gesin.

Java

In the Java track, students explore back-end coding through learning Java and gaining a deep understanding of its computer science concepts. Students use an array of software including BlueJ, Android Studio, and Apache Netbeans to learn the programming language. Through the junior and senior classes, students have the opportunity to program complete playable games of their own design under the guidance of their teacher, Ms. Gesin.

Students take a marking period of an Introduction to Programming, where they learn the basics of Python and, for many, are exposed to computer science for the first time.

The Java Programming elective teaches students the basics of Java. This education is reinforced through problem-solving labs and a marking period-long project in which students create their own text-based game.

The Advanced Java elective further challenges students to use their Java skills on a modern, realistically professional level. More complex concepts are taught and utilizied to develop more complicated and demanding programs. Further, each year, students of this class code their own original app in Android Studio and submit their creations to the Congressional App Challenge.

Web Design

Web design focuses on the back-end coding of websites. Students learn HTML/CSS, which is the structure of a website as well as its design elements. Students explore current websites and analyze and learn the importance of their usability.

Students take a marking period of an Introduction to Programming, where they learn the basics of Python and, for many, are exposed to computer science for the first time.

The Web Design & Digital Literacy elective introduced students to HTML. After building skeletons of websites, CSS is introduced. Students are able to create their own websites for their interests. Additionally, students read and learn about the history of the internet and its impact today. By the end of the semester long class, students are given a pretend client to build a website for to mimick real world experience.

The Advanced Web Design pushes students to create website centered around user experience and accessibility. Using the tools learned in the previous class, students apply new concepts of usability and common web conventions to code a modern, accessible website. Students further their knowledge by creating responsive designs, functioning from desktop to mobile. Additionally, PHP is introduced for students to begin exploring.
